
-- Interaction Design.
- Se focaliza en como crear interacciones eficientes e interactivas entre el usuario y las interfaces.
Estas interactiones pueden suceder dentro de las pantallas, entre las pantallas y más allá de las pantallas.

- Se puede diseñar todo lo que sucede tantdo dentro de la pantalla como la interacción del usuario o los 
eventos de pantalla, también se pueden agregar pequeñas líneas de código que den referencia de eventos.

- También se pueden agregar elementos zoom para mostrar comportamientos desde cerca.

- Otro comportamiento común a mostrar son las paths de una pantalla a otra pantalla.

- LA IDEA ES QUE EL WIREFRAME INTERACTIVO MUESTRE TODA LA FUNCIONALIDAD COMPLETA ANTES DE QUE EL DESARROLLADOR 
EMPIECE A TRABAJAR.

- También se realizan anotaciones pertinentes sobre funcionamientos y eventos.

- Es posible que también se requiera hacer algo de User Research si no hay alguien dedicado en el equipo.
